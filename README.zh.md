## 快速开始

```bash
# 安装
npm install -g oh-my-copilot

# 配置
omp setup
```

## 功能

- **多代理编排** - 18 个专业代理处理不同任务
- **MCP 服务器集成** - 连接外部服务和工具
- **HUD 显示** - 实时状态和上下文跟踪
- **Plugin State Manager** - 跨会话的可靠状态管理
- **30+ 技能** - 扩展功能的延迟加载
