# SciOMC

> **OMG adapted scope:** The name stays `sciomc` for upstream discoverability, but this is an OMG-adapted degraded port. Use 3–5 bounded analysis facets, not 20-way parallel parity.

## Workflow
1. Define the research question, available data, constraints, and success criteria.
2. Split into 3–5 independent analysis facets at most.
3. Run or delegate each facet only when it has a distinct hypothesis or method.
4. Compare findings, contradictions, and confidence.
5. Produce a final synthesis with evidence, limitations, and next experiments.

## Rules
- Do not claim statistical certainty without data and method evidence.
- Keep concurrency expectations explicit and bounded.
- Prefer reproducible commands or notebooks for data work when available.
