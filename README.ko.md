## 빠른 시작

```bash
# 설치
npm install -g oh-my-copilot

# 설정
omp setup
```

## 기능

- **멀티 에이전트 오케스트레이션** - 다양한 작업을 위한 18개의 전문 에이전트
- **MCP 서버 통합** - 외부 서비스 및 도구 연결
- **HUD 디스플레이** - 실시간 상태 및 컨텍스트 추적
- **Plugin State Manager** - 세션 전반의 신뢰할 수 있는 상태 관리
- **30+ 스킬** - 확장 기능의 지연 로딩
